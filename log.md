# AI 工具集合站 — 更新日志

## V1.1 (2026-06-22)

### 🚀 新增功能

#### 支付系统 (P0)
- 微信支付 Native V3 接入（扫码支付）
- 支付宝当面付接入（扫码支付）
- 统一支付接口 `src/lib/payment.ts`，通过环境变量切换渠道
- BuyButton 组件：一键创建订单 → 二维码弹窗 → 支付确认
- 支付回调自动处理：验签 → 标记已支付 → 发放次数/升级会员 → 邀请奖励
- 未配置支付渠道时优雅降级为开发模式

#### 邀请裂变 (P0.5)
- 新用户注册自动生成 8 位邀请码
- 邀请链接：`/auth/signin?invite=XXXX`
- 奖励规则：邀请 1 人注册 +10 次，邀请 1 人付费 +50 次
- Dashboard 邀请页面：专属链接复制、已邀请用户列表
- 仪表盘首页增加邀请统计卡片

#### SEO 内容扩充 (P1)
- 模板库：7 → 30 个，新增 企业管理、SaaS 两大分类
- 博客文章：2 → 20 篇，覆盖 11 个标签分类
- 博客分类页 `/blog/category/[slug]`：按标签筛选文章
- 博客首页动态生成文章列表 + 分类导航
- Sitemap 增加博客分类路由

#### 图表分享 (P2)
- 公开分享页 `/tools/share/[id]`：无需登录即可查看图表
- SEO 友好：动态 meta 标签（标题、描述）
- 分享页底部 CTA 引导用户注册

#### 游客体验优化 (P2)
- 游客现在可免费使用 AI 生成 3 次（之前为 0）
- 游客导出次数调整为 1 次
- 按 deviceId 追踪游客 AI 使用量
- 次数用完提示注册（注册送 30 次）

#### Mermaid 在线编辑器 (P3)
- 安装 Monaco Editor (`@monaco-editor/react`)
- 左侧代码编辑 + 右侧实时预览（500ms 防抖）
- 支持从历史记录加载图表
- 保存到历史、下载 SVG
- 图表类型切换（6 种）

### 🔧 数据库变更
- `Subscription` 模型：独立管理会员订阅状态和到期时间
- `Order` 模型增加：`tradeNo`, `productId`, `credits`, `paidAt`, `expireAt`
- `User` 模型增加：`inviteCode`, `inviterId`（自关联）
- `GuestUsage` 模型增加：`aiCount`（游客 AI 使用追踪）

## V1.0 (2026-06-21)

### 初始版本

#### 核心功能
- AI 图表生成：支持流程图、ER图、UML、时序图、甘特图、思维导图 6 种类型
- Mermaid 渲染：客户端 SVG 渲染 + 代码复制 + SVG 下载
- 用户认证：NextAuth Email Provider 魔法链接登录
- 会员体系：游客/免费/Pro 三级，次数控制
- 用户中心 Dashboard：首页/历史记录/收藏夹/会员中心/账号设置

#### 内容系统
- 模板库：7 个预设模板（学生管理、电商系统、库存管理、医院管理、博客系统）
- SEO 博客：2 篇完整文章（ER图教程、UML教程），配置驱动可扩展
- Sitemap + Robots.txt
- 响应式设计：桌面端 + 移动端

#### 技术栈
- Next.js 14 (App Router) + TypeScript + Prisma (SQLite)
- NextAuth v4 + Mermaid.js v11 + Tailwind CSS
- DeepSeek/OpenAI 兼容 API

#### 统计
- 33 页面路由 / 9 API 路由 / 7 组件 / 6 数据库模型
- ~6,000 行代码
